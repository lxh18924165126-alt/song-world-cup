# 项目摘要

## 一句话定义
把用户的 QQ 音乐或网易云音乐公开歌单转成“世界杯淘汰赛”，通过逐场对决选出冠军歌曲，并在完赛后生成结果页、赛后分享链接与带公开赛果二维码的超大画布对阵图。

## 第一版目标
- 完整跑通：导入歌单 → 歌曲检查 → 赛事设置 → 抽签预览 → 正式比赛 → 完赛总结 → 赛后分享；
- 优先保障移动端体验；
- 匿名模式可完整使用，微信与 QQ 登录代码要完成；
- Cloudflare 仅使用免费套餐；
- 导出图全部由用户设备本地生成并直接下载。

## 第一版明确范围
- 一届赛事只支持导入一个 QQ 音乐或网易云音乐公开歌单；
- 规模支持 16/32/64/128/256/512/1024/2048/4096 与“所有歌曲随机”；
- 固定左右赛区，最终汇聚总决赛；
- 比赛使用固定超大签表画布和受控镜头移动，八强、半决赛与总决赛在同一画布中逐步强化舞台表现；
- 第一版不做中途分享，不做公开赛事广场。

## 冻结决策
- 正式开赛后不允许重开、不允许复制副本、不允许覆盖重抽；
- 不设置季军赛；
- 不使用单曲封面；
- 完赛后默认不生成分享链接，只有创建者主动“开放赛后分享”或下载带二维码的对阵图时才生成；
- 分享页只展示歌单名称和歌单封面，不公开原音乐平台歌单链接；
- 分享页完全不提供媒体按钮。
