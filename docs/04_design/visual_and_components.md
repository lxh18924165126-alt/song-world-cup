# 视觉风格与组件规范

## 视觉方向
- 暗黑竞技转播风；
- 背景深黑、炭灰；
- 主强调为竞技红；
- 冠军与奖励使用金色；
- 依靠聚光灯、奖杯、对阵线与信息层级建立仪式感；
- `docs/00_overview/project_summary.md` 规定不使用单曲封面；视觉层以排版、聚光灯、奖杯与对阵线建立辨识度。

## 设计令牌
- 背景：`#090B0E / #11161C / #171D24`
- 强调红：`#E53935`
- 强调金：`#F5C46B`
- 辅助蓝：`#4DA3FF`
- 主文字：`#F5F7FA`
- 次文字：`#AAB4C0`
- 常用圆角：`16 / 20 / 24`

## 关键组件
- 歌曲卡片：歌名、歌手、媒体按钮、选中态、禁用态；
- 对决行：左卡片 / VS / 右卡片；
- 轮次头部：赛事名称、赛区、轮次、进度；
- 抽签预览卡：赛制摘要、轮空数、签表缩略图；
- 结果摘要卡：冠军、亚军、四强、统计、分享与导出；
- 我的赛事列表项：状态标签、信息、操作按钮；
- 后台统计卡：指标值、健康状态、快捷操作。
